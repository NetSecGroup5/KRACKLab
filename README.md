Ex2 (Xubuntu 24.04.4 minimal)

- clone the lab repo with git clone --recurse-submodules https://github.com/NetSecGroup5/KRACKLab.git in the Documents folder
- go to KRACKLab/lab/ex2/krackattacks-script/krackattack/
- run build.sh and pysetup.sh (require to install packages libnl-3-200 libnl-3-dev libssl-dev python3.12 venv)
- move to KRACKLab/lab/ex2/krackattacks-script/hostapd and verify that there is hosatpd file. If not, run make
- go to KRACKLab/lab/ex2/mininet-wifi/
- run sudo util/install.sh -Wlnfv
- Move to Downloads folder
- Obtain wpa_supplicant 2.3 source code with wget https://w1.fi/releases/wpa_supplicant-2.3.tar.gz
- Extract it with tar -xvzf wpa_supplicant-2.3.tar.gz wpa_supplicant-2.3/
- Exe cp defconfig .config
- Change the following line in the .config just created:
    - Uncomment CONFIG_LIBNL32=y
    - Uncomment CONFIG_TLS=openssl and set it to CONFIG_TLS=internal
- Compile executing the command make (requires package libtommath-dev)
- Copy executable wpa_supplicant to KRACKLab/lab/ex2/ and rename it in wpa_supplicant23
- Install packages xclip wireshark
- Open wireshark with sudo, go to Edit > Preferences > Protocols > IEEE 802.11 > Decryption keys > new field > Key Type select wpa-pwd > key value set to abcdefgh:testnetwork. You will be able to set the CCMP Packet Number column after starting receiving packets. You need to select a packet, select IEEE 802.11 QoS Data in the package info > CCMP parameters > CCMP Ext. Initialization Vector
- Move to Downloads folder
- Download linux kernel 4.0 (fix for KRACK attack introduced in later version) with:
    - wget https://kernel.ubuntu.com/mainline/v4.4-wily/linux-headers-4.4.0-040400-generic_4.4.0-040400.201601101930_amd64.deb
    - wget https://kernel.ubuntu.com/mainline/v4.4-wily/linux-image-4.4.0-040400-generic_4.4.0-040400.201601101930_amd64.deb
- Install with sudo dpkg -i linux*.deb
- Change grub config in /etc/default/grub
    - Delete GRUB_TIMEOUT_STYLE=hidden
    - Set GRUB_TIMEOUT=0 to GRUB_TIMEOUT=10
    - Delete GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
- Run sudo update-grub
- Restart and on grub select other options > linux-kernel-4.4.0-generic
- Run ex2.py with sudo python3 ex2.py (remember exe chmod +x ./startAttack.sh before start)
